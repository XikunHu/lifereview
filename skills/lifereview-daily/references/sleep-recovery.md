# 睡眠与恢复指南

## 睡眠误判保护

深夜（23:00-06:00）若 Looki moment 描述同时含「屏幕亮 + 卧室/睡眠环境词」，按睡眠处理，不计入活动跨度。

## 睡眠建议生成规则

1. 日历 >8h + 最晚会议 >21:00 → 警告"认知激活风险"，建议 30-45min 切离期
2. RHR > 基线+5 → "自主神经应激"，建议睡前 5-4-7 呼吸法
3. 昨晚睡眠 <6h → 强烈建议今晚 ≥7.5h，明天不排 9:00 前会议
4. HRV 白天有明显低谷（<50ms）→ 标注压力峰值，建议该时段后主动恢复
5. RHR 日间波动 >20bpm 或有饮酒记录 → 今晚不喝酒
6. RHR <50 且 HRV >65 且睡眠 ≥7h → 正面强化：身体扛住了

## 相关因果规律

- 酒精 → RHR +5~+6 bpm（基因层面确认慢代谢型——自行做基因检测验证）
- 深睡翻倍 ≠ 睡得好——是白天透支的维修账单
- 睡前 HR 峰值 → 深睡比例 r=+0.70（p<0.01）
- 睡前 HRV → 深睡比例 r=-0.87（p<0.01）
